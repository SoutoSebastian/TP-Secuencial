# TP-Secuencial

Implementación de programas del libro "Sequential Methods and Their Applications" Nitis Mukhopadhyay y Basil M. de Silva

## Programa 1. Power (Cap 2, Sección 2.5)
Cálculo de probabilidad de error de tipo II o cálculo de tamaño muestral para el test de Neyman-Pearson.

## Programa 2.  ProbKn (Cap 2, Sección 2.5)
Analizar la longitud aleatoria de un intervalo de confianza para la media de una población Normal.

## Programa 3. SPRT (Cap 3, Sección 3.2)
Realizar un test secuencial de hipótesis simples sobre un parámetro θ.

## Programa 4. OC (Cap 3, Secciones 3.3 y 3.4)
Analizar el comportamiento del SPRT para diferentes valores verdaderos del parámetro θ.

## Programa 5.  Stein Two-Stage (Cap 6, Sección 6.2)
Construir un intervalo de confianza de longitud fija 2d para la media µ de una población Normal.
